# Thai Financial Holiday - Demo & Docs

เว็บไซต์ Demo & Documentation สำหรับแพ็กเกจ [@krizad/thai-financial-holiday-helper](https://www.npmjs.com/package/@krizad/thai-financial-holiday-helper)

## 🌟 Features

- **Interactive Playground** - ตรวจสอบวันหยุด/วันทำการแบบ real-time
- **Holiday Calendar** - ปฏิทินวันหยุดกรองตามปี/เดือน
- **Business Days Calculator** - คำนวณวันทำการ (บวก/ลบ + ช่วงเวลา)
- **API Documentation** - ตัวอย่างโค้ดครบทุกฟังก์ชัน

## 🛠️ Development

```bash
npm install
npm run dev
```

## 📦 Build
